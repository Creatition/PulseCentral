About PulseCentral

Your open-source PulseChain portfolio tracker.

🚀 What is PulseCentral?

PulseCentral is a free, open-source portfolio and market tracker for PulseChain — the EVM-compatible blockchain launched by Richard Heart. Inspired by plsfolio.com, PulseCentral lets you:

  • Track any PulseChain wallet's token holdings and total USD value.
  • Browse live market prices for popular PulseChain tokens.
  • Discover trending tokens on PulseChain DEXes.

⛓ Data Sources

  • Wallet balances — PulseChain Scan API (scan.pulsechain.com)
  • Token prices & DEX data — DexScreener API (dexscreener.com)

📋 Upcoming Features

  • Multi-wallet aggregation
  • Historical portfolio chart
  • NFT tab
  • DeFi / farming positions tab
  • Price alerts
